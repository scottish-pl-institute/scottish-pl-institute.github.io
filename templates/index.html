SPLS

The Scottish Programming Languages Seminar (SPLS) is a forum for discussion of all aspects of programming languages. We meet for a day or afternoon once every few months, at some congenial location in Scotland. Talks by Scots and by visitors are welcome.

SPLV

The Scottish Programming Languages and Verification Summer School (SPLV) provides core and advanced classes that cover foundational as well as state of the art knowledge on the underlying principles needed to use or design new languages and verifying program correctness.

Zulip

The Scottish Programming Languages Zulip instance facilitates programming languages-related discussion amongst SPLI members.
